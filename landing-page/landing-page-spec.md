This landing page is designed specifically for Google Ads and SEO campaigns targeting the keyword "קייטרינג כשר" (Kosher Catering). It prioritizes relevance, transparency, and action, adhering to the "Agada" brand identity while using the specific sales copy provided.

3. Information Architecture & Layout (Wireframe Logic)

1. Sticky Mobile Header / Floating CTA

Requirement: As per the document ("שים כפתור 'חייג עכשיו' שצף ורואים אותו כל הזמן"), a sticky bottom bar or floating button on mobile is mandatory.

Elements: Phone Icon + "Call Now" text.

2. Hero Section (Above Fold)

Layout: Centered text over high-quality background image (Chef/Meat dish).

Visual: Dark overlay to ensure text readability (#070707 w/ opacity).

Elements: H1 -> H2 -> Primary CTA Button.

3. "The Problem" (Text Block)

Layout: Single column text, centered or justified.

Visual: Light text on dark background (Contrast).

4. Features Grid (Why Agada)

Layout: 2 or 3 columns on Desktop, 1 column on Mobile.

Visual: Icon (Gold/Green) above bold Title. Text below.

5. Menu Teaser (Visual List)

Layout: Split layout (Text on one side, Appetizing image on the other) OR 3 Cards (Starters, Mains, Sides).

Visual: High-quality food photography.

6. "You Deserve" (Checklist)

Layout: Bulleted list with Checkmarks.

Visual: Clean typography, perhaps a different background shade (Dark Grey #111111) to separate sections.

7. FAQ Accordion

Layout: Collapsible questions to save vertical space.

Behavior: Click to expand answer.

8. Final CTA Footer

Layout: Bold background color (#16A34A), Centered text, Large Phone Number.

4. Technical Requirements

Performance: Images must be optimized (WebP format) for fast mobile loading.

Responsive: Mobile-first design philosophy.

Accessibility: High contrast text, clear tap targets.
